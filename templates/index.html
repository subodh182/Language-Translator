About LinguaFlow

LinguaFlow is a professional language translation web application built with Flask and powered by Google's neural translation engine.

  • ✦ 100+ supported languages
  • ✦ Auto language detection
  • ✦ Text-to-speech playback
  • ✦ Real-time auto-translation
  • ✦ One-click copy & paste

Built with Flask · Python · Vanilla JS
